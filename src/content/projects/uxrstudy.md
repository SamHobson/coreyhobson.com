UXR Study is a digital storefront selling premium Notion templates for UX researchers. Built for Corey's wife, the site showcases and sells research templates through Gumroad integration. Each product page features previews, feature lists, and direct purchase links.

The site runs on a modern Jamstack architecture with Sanity providing the content backend — template descriptions, pricing, feature lists, and changelogs are all managed through Sanity Studio. The frontend is fully static, exported at build time for maximum performance on Vercel's edge network.

## Stack

- **Framework:** Next.js 16 (App Router)
- **CMS:** Sanity v5 with structured content modeling
- **Styling:** Tailwind CSS v4
- **Hosting:** Vercel (static export)
- **Payments:** Gumroad integration
- **Package manager:** pnpm
- **Typography:** Poppins (Google Fonts)
- **Color:** Navy palette (#112236 to #061843)

## Products

Nine research templates are currently live, covering user interviews, usability testing, competitive analysis, and synthesis workflows. Each template ships as a Notion duplicatable link with setup instructions.
